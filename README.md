#  Presentation Maker

Веб-приложение для создания и редактирования презентаций.

## 🚀 Технологии

- **React** — компонентный подход, хуки (useState, useEffect)
- **TypeScript** — статическая типизация
- **Vite** — быстрая сборка и HMR (Hot Module Replacement)
- **CSS Modules** — изоляция стилей компонентов

## 🎯 Функционал

- Создание слайдов презентации
- Редактирование контента
- Навигация между слайдами
- Современный UI/UX

## 📦 Установка и запуск

```bash
# Установи зависимости
npm install

# Запусти проект (режим разработки)
npm run dev

# Собери для продакшена
npm run build

# Preview собранной версии
npm run preview
